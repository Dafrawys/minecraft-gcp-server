\# Minecraft Server on GCP 🚀



Automated deployment of a Minecraft server on Google Cloud Platform using:



\- Bash (quick deploy)

\- Terraform (production-grade)



\---



\## ⚙️ Requirements



\- gcloud CLI

\- Terraform

\- GCP project with billing enabled



\---



\## 🚀 Quick Start (Bash)



```bash

cp .env.example .env

chmod +x deploy.sh

./deploy.sh
